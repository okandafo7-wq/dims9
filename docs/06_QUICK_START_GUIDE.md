## Common Tasks

### Log Production Data

1. Navigate to "Data Entry"
2. Enter date and measurements
3. Select energy usage
4. Add issues if any
5. Click "Submit"

**Tip:** Log data daily or weekly consistently for best insights.

---

### Manage Issues

**Create Issue:**
1. Go to "Issues" page
2. Click "Add Issue"
3. Select category and severity
4. Write description (include ISO clause)
5. Document corrective action
6. Assign to team member
7. Click "Create"

**Update Status:**
1. Find issue card
2. Click status button:
   - "Start Working" (Open â†’ In Progress)
   - "Mark as Closed" (In Progress â†’ Closed)
   - "Reopen" (Closed â†’ Open)

**Edit Issue:**
1. Click "Edit" button on issue
2. Update fields
3. Click "Update Issue"

---

### Run What-If Simulation

1. Navigate to "Simulator"
2. Current data pre-filled
3. Adjust sliders:
   - Loss reduction target (%)
   - Investment amount ($)
4. View results:
   - Additional production
   - Revenue increase
   - ROI percentage
   - Payback period
5. Use insights for decisions

---

### View ISO Compliance

1. Go to "ISO Compliance" page
2. See overall score
3. Click tabs for each standard:
   - ISO 9001 (Quality)
   - ISO 14001 (Environment)
   - ISO 45001 (Safety)
4. Review requirements and issues
